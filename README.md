# Fishy 🐟

Een onderwater "eet of word gegeten" spel. Bestuur de happende vis, eet kleinere
vissen om te groeien en ontwijk de grotere.

- **Mobiel:** sleep met je vinger (landscape / horizontaal spelen)
- **Desktop:** pijltjestoetsen ← ↑ ↓ →

Pure HTML5 + canvas, geen build nodig. Open `index.html` of speel de gehoste versie.

## Bestanden
- `index.html` — het complete spel
- `frames/` — de geanimeerde speler-sprite (frames uit een video, achtergrond weggeknipt)
- `cutout.swift` / `cutout_batch.swift` — de Vision-scripts die de achtergrond per frame verwijderden
